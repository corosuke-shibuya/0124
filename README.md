# ①課題名 
ブックマーク管理アプリ（ログイン認証／権限管理つき）

## ②課題内容（どんな作品か） 
- 書籍名・URL・コメントを登録し、一覧で閲覧できるブックマーク管理アプリです。
- ブックマークの一覧を見るためにログイン機能（ID/PW）を実装
- manager権限は更新・削除可能、member権限は閲覧のみにしました
  
## ③アプリのデプロイURL デプロイしている場合はURLを記入（任意） 
https://corosuke-shibuya.sakura.ne.jp/0124/

## ④アプリのログイン用IDまたはPassword（ある場合） 
- ID: manager
- PW: manager1
- ID: member
- PW: manager1

- ## ⑤工夫した点・こだわった点 
- パスワードは password_hash() / password_verify() を用いてハッシュ化し、平文保存しない形にしました。
- kanri_flg による権限分岐を実装し、管理者のみ「削除・編集」リンクが表示され、実行もできるようにしました。
  
## ⑥難しかった点・次回トライしたいこと（又は機能） 
- さくらサーバー上での反映遅延／VSコードで別ファイルを更新していることに気づかないなどが原因で思うように反映されない場面があり、原因切り分けに苦戦しました
- ハッシュ化が理解できておらず、苦戦しました
- 次回は登録画面を作って、自動でハッシュ化されるようにしたいです
